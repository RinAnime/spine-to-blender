# spine-to-blender

### 使用
- Spine Json 版本 : 3.8
- Blender 版本 : 3.6+

- Blender偏好设置 > 插件 > 安装 [ Spine_To_Blender.py ]

- 插件面板选择对应的文件目录 .json / .atlas / .png , 点击 [ 导入 ]

### 注意
- 附件(Attachments)类型为"region", 并且使用的是解包图集的图片
  则选择图集文件夹并使用 <Re:Dive> 类型导入

- 关于 spine-json-format: http://esotericsoftware.com/spine-json-format
